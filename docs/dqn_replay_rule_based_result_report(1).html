DQN Replay 实验结果报告

Label method: rule_based | Methods: online, uniform, per, regime | Seeds: 0, 1, 2

1. 实验设置概述

本次实验使用 rule_based regime labels，对四组 DQN 方法进行对比：

方法含义主要用途
onlineDQN-only / no replay不使用 replay buffer，只用最新 transition 更新，作为无 replay 基线。
uniformDQN + Uniform Replay从历史 buffer 中等概率采样，用于观察普通 replay 是否混入大量其他 regime 样本。
perDQN + Prioritized Experience Replay按 TD-error 优先采样，用于检验 TD-error priority 是否会偏向旧 regime 高误差样本。
regimeDQN + Regime-aware Replay按当前 regime、TD-error、recent、random 的混合规则采样，用于降低 regime mismatch。
本次结果的核心问题：普通 replay 与 PER 是否存在明显 regime mismatch；regime-aware replay 是否能降低 mismatch；这些机制变化是否已经转化为更好的交易表现。

4. 结果解读

4.1 交易表现：所有方法都还不是有效交易策略

四组方法的最终净值均明显低于 1，平均最大回撤都在 0.93 到 0.96 左右。这说明当前实验主要用于验证 replay 机制，而不是证明 DQN 已经能形成有效交易策略。

  • DQN + Uniform Replay 在平均最终净值和平均最大回撤上表现最好：final value ≈ 0.0759，max drawdown ≈ 0.9357。
  • DQN-online 无 replay，最终净值最低之一，说明只用最新样本更新的 sample efficiency 较差。
  • DQN + Regime-aware Replay 虽然降低了 mismatch，但交易表现还没有显著超过 Uniform Replay。

4.2 Replay 机制：Regime-aware 明显降低 mismatch

Uniform Replay 的平均 mismatch rate 为 0.5674，PER 为 0.5785，Regime-aware Replay 降至 0.2772。

这意味着 Regime-aware Replay 相比 Uniform Replay 将 mismatch rate 降低约 51.1%，相比 PER 降低约 52.1%。

机制结论：Regime-aware Replay 达到了设计目标：它确实更少从与当前 regime 不一致的历史 transition 中采样。这支持后续继续研究 regime-aware sampling。

4.3 PER：TD-error 更高，但 mismatch 也更高

PER 的平均 TD-error 为 0.0138，高于 Uniform Replay 的 0.0097。这说明 PER 确实更偏向高 TD-error 样本。

但 PER 的 mismatch rate 也高于 Uniform：0.5785 vs 0.5674。这说明高 TD-error 样本不一定来自当前 regime，可能包含更多旧 regime 或不兼容 regime 的 transition。

研究含义：在金融非平稳环境中，TD-error priority 可能并不是可靠的 replay relevance signal。它可能提升样本“误差强度”，但不一定提升样本与当前 regime 的相关性。

4.4 Regime-aware：机制有效，但交易收益仍需调参或改进

Regime-aware Replay 的 mismatch rate 最低，但 final value 只略高于 PER，低于 Uniform Replay。这说明当前 sampler 在“降低错配样本”上有效，但还没有把机制优势稳定转化为更好的 portfolio return。

可能原因包括：same-regime 比例过高导致样本多样性不足；交易成本和高换手对净值侵蚀较大；DQN 的离散动作空间较粗；当前结果是在线训练期表现，不是严格 train/test 分离后的回测表现。

6. 当前可写入报告的结论

建议表述：

Using rule-based regime labels, we compare DQN-online, DQN with uniform replay, DQN with PER, and DQN with regime-aware replay across three random seeds. The results show that standard replay methods suffer from substantial regime mismatch: Uniform Replay has an average mismatch rate of 0.5674, while PER has an even higher mismatch rate of 0.5785. The proposed regime-aware sampler reduces the mismatch rate to 0.2772, achieving about 51.1% reduction relative to Uniform Replay. However, this reduction does not yet translate into the best trading performance: Uniform Replay still achieves the highest average final portfolio value and the lowest average maximum drawdown. This suggests that regime-aware replay successfully addresses the replay contamination mechanism, but further tuning is needed to improve trading performance.

下一步建议

  1. 对 regime_same_ratio 做 sensitivity test，例如 0.3 / 0.4 / 0.5 / 0.6，检查是否存在更优的 regime-aware sampling ratio。
  2. 降低交易成本或降低探索期换手，例如缩短 epsilon_decay_steps，观察 DQN 净值是否明显改善。
  3. 分别运行 hmm 与 recap_cusum labels，对比不同 regime labeling 对 mismatch rate 和交易表现的影响。
  4. 增加严格 train/test 分离，避免把训练期 portfolio value 当成正式回测表现。
